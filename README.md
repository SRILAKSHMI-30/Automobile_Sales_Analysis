# 🚗 Automobile Sales Analysis Dashboard

## 📌 Overview
This project provides a detailed **automobile sales performance analysis** through an interactive dashboard.  
It consolidates metrics such as **profit, quantity sold, unit price, revenue, and ratings**, along with regional and state-level breakdowns.  
The dashboard enables quick identification of sales trends, regional performance, and profitability drivers.

## 📈 Insights
- **Peak Profit Year:** 2025 with 0.76M profit.  
- **Regional Leader:** West region leads in both unit price and quantity sold.  
- **State Highlight:** Kentucky tops profit, while New Jersey leads in customer ratings.  
- **Trend:** Profit shows growth until 2025, followed by a decline in 2026.  
- **Balanced Sales:** Categories A–D show relatively even distribution across regions.
